# SmartCity_trashcans

## 개발 동기
- 넘쳐나는 쓰레기, 악취 등으로 인한 미화 측면에서의 불편함
- 환경 미화원분들의 번거로움 감소

## 목표
- 자원 절약
- 인력 절약
- 미관 향상
- 악취 감소
